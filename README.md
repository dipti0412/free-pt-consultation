# Free PT Consultation

## MVP Features

- Suggest weekly/daily workout plans based on user history.
  - Next version: automatically update plans when workouts are missed.
- Suggest weekly/daily nutrition plans.
  - Next version: improve meal plan recommendations based on goals, regional food preferences, and personal choices.

## Input

- Garmin data
- Smart weighing scale data
- Goals
- Intake form details, including health issues and blood test reports

## Output

- Weekly workout plan
- Weekly nutrition plan
- Sleep needs
